# Go-Kart Chassis Design and Structural Analysis

## Overview

This project focuses on the design and structural analysis of a tubular Go-Kart chassis using SolidWorks and ANSYS. The chassis was modeled using SolidWorks weldment features and analyzed under a front impact loading condition to evaluate its structural performance. The analysis provides insights into stress distribution and deformation, helping assess the chassis' ability to withstand applied loads.

---

## Objectives

- Design a lightweight tubular Go-Kart chassis.
- Create a 3D CAD model using SolidWorks.
- Perform static structural analysis in ANSYS.
- Evaluate stress distribution and deformation under front impact loading.
- Assess the structural integrity of the chassis.

---

## Software Used

- **SolidWorks 2021**
- **ANSYS 2025 R2 Student**

---

## Project Workflow

1. Designed the Go-Kart chassis using SolidWorks weldment features.
2. Exported the CAD model in STEP format.
3. Imported the geometry into ANSYS.
4. Generated the finite element mesh.
5. Assigned Structural Steel material properties.
6. Applied boundary conditions and front impact loading.
7. Performed static structural analysis.
8. Evaluated Total Deformation and Equivalent (von Mises) Stress results.
